# Introsphere - Opis Projektu

## One-Liner
Aplikacja do prowadzonej introspekcji dla młodych dorosłych (Gen Z) - 7-dniowe "journeys" z codziennymi pytaniami + kalendarz nastrojów.

## Kluczowe Funkcje

### 1. Wybór Atmosfery (Krok 1)
- Użytkownik wybiera "atmosferę" / "vibe" aplikacji
- 5 atmosfer podstawowych (free):
  1. **Clean Girl Aesthetic** - calm, minimal, beige/cream/soft neutrals, simple UI, lots of white space. Dla osób skupionych na wellness i rutinach.
  2. **Cottagecore** - slow life, nature, greens/warm browns, flowers, handwritten fonts. Romantyzacja prostego życia.
  3. **Dark Academia** - introspective, intellectual, dark brown/burgundy/parchment, typewriter fonts. Romantyzacja literatury.
  4. **Soft Girl Aesthetic** - emotional openness, pastel pink/lavender/baby blue, cute icons. Dla emocjonalnego wyrażania.
  5. **Futuristic Femme** - tech-wellness, chrome/silver, sleek UI, glowing gradients. Dla fanów technologii.
- Wybór wpływa na UI/tła aplikacji
- Gen Z lubi personalizację "look & feel"

### 2. Wybór Trybu: Journey lub Jednorazowa Sesja
- **Opcja A: Guided Journey** - kilkudniowy program (np. 7 dni)
- **Opcja B: Jednorazowa sesja** - pojedyncze pytanie

### 3. Wybór Emocji (Koło Emocji)
- **Krok A:** Wybór ogólnej emocji (np. radość, smutek, złość, strach, wstręt, zaskoczenie)
- **Krok B:** Opcja precyzacji - rozwinięcie emocji (np. smutek → tęsknota, samotność, żal)
- **Krok C:** Lub zostań na ogólnej emocji
- **Krok D:** Z banku pytań losowane jest pytanie dla wybranej kategorii emocji

### 4. Odpowiedź
- Formularz tekstowy
- Zapis do Firebase / localStorage

### 2. Kalendarz Nastrojów
- Śledzenie nastrojów w czasie
- Wizualna reprezentacja (kolorowy kalendarz)
- Statystyki: najczęstsze nastroje, trendy

### 3. Baza Pytań Shadow Work
- Przygotowane pytania do pracy z cieniem
- Kategorie: relacje, samopoczucie, kariera, przeszłość, lęki, radości

## Technologie (do użycia)
- **Frontend:** React + Tailwind CSS + Shadcn
- **Backend:** Firebase lub Supabase
- **Hosting:** Vercel (darmowe dla hobby)

## Limit Czasowy
- **Projekt:** 15 godzin
- **Poziom:** Początkująca (HTML, CSS, troszke JS)
- **Podejście:** MVP - minimum funkcji

## Monetyzacja (SaaS)
- **Freemium:** 3 sesje/miesiąc, podstawowy kalendarz
- **Premium (9-15$/mc):** Nieograniczone sesje, rozszerzony kalendarz, eksport danych

## Dlaczego to jest niszowe?
- Prowadzone journeys = unikalne (nikt nie ma tego w tej formie)
- Rynek wellness dla Gen Z rośnie
- Konkurencja: głównie ogólne aplikacje do medytacji (Headspace, Calm) - nie ma nic specyficznie dla prowadzonej introspekcji z artystycznym elementem
